Below are the given arrays:

const people = [ { name: "Wes", year: 1988 }, { name: "Kait", year: 1986 }, { name: "Irv", year: 1970 }, { name: "Lux", year: 2015 }, ];

const comments = [ { text: "Love this!", id: 523423 }, { text: "Super good", id: 823423 }, { text: "You are the best", id: 2039842 }, { text: "Ramen is my fav food ever", id: 123523 }, { text: "Nice Nice Nice!", id: 542328 }, ];

Questions:

• Some and Every Checks
1. Array.prototype.some() : is at least one person 19?

2. Array.prototype.every() : is everyone 19?


• Array.prototype.find()
Find is like filter, but instead returns just the one you are looking for
3. Find the comment with the ID of 823423


• Array.prototype.findIndex()
Find the comment with this ID
4. Delete the comment with the ID of 8234

Below are the answers. Take a look at the Javascript Console for result 

   1.
                const isAdult = people.some(
                    (person) => new Date().getFullYear() - person.year >= 19
                );
              
   2.
                const allAdult = people.every(
                    (person) => new Date().getFullYear() - person.year >= 19
                );
              
   3.
                const comment = comments.find((comment) => comment.id === 823423);
              
   4.
                const index = comments.findIndex((comment) => comment.id === 823423);
                const newComments = [
                    ...comments.slice(0, index), 
                    ...comments.slice(index + 1)
                ];
